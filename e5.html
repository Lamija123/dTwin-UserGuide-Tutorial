5.12. State estimation with Equality Constraints

Task Overview:

This task implements a weighted least squares (WLS) state estimation algorithm for a 3-node power network. The model integrates both measurement equations and equality constraints, resulting in enhanced estimation accuracy and system observability.

Task Objective:

The goal is to determine the complex voltage phasors (v₁, v₂, v₃) at all nodes of the network using:

  • Magnitude measurements of voltages at nodes 1, 2 and 3,
  • Complex power injection at node 3,
  • Slack node angle constraint for node 1,
  • Kirchhoff’s Current Law (KCL) constraint for node 2.

Explanation of Key Components

  • WLSEs (Weighted Least Squares Equations) include:
    • Slack bus angle: fixed using complex conjugate equality on v₁,
    • Voltage magnitudes: enforced at nodes 1, 2, and 3,
    • Power injection: specified at node 3,
  • ECs (Equality Constraints):
    • Node 2 is a zero-injection node (ZI); current balance is enforced via KCL.
