Allowed mathematical functions - examples

Real Functions

The following functions are permitted in models using real numbers:

  • abs

    • The function is used to calculate the absolute value of x.

    An example of using this function is given below.

    Model [type = NL, name = "examle abs", domain=real, eps=1e-6, method=NR]: Vars [out=true]: y1 y2 Params: x1=5 x2=-5 NLEs: abs(x2)=y2 abs(x1)=y1 end //end of model

    • Functions are used to calculate mathematical functions arccos(x), arcsin(x) and arctan(x) in radians.

  • min, max, smin, smax, avg, var, dev

    • The functions min and max determine the smallest and largest value between two numbers, smin and smax represent their smoothed (differentiable) versions, while avg calculates the mean value of a set of numbers, and var and dev compute the variance and standard deviation of that set.

    // Model demonstrating min, max, smin, smax, avg, var, dev Model [type = NL, name = "test_statistical_functions", domain = real, eps = 1e-6, method = NR]: Vars [out=true]: result_min result_max Params: a = 5 b = 8 NLEs: // Basic min/max result_min = min(a, b) result_max = max(a, b) end // end of model

  • diff, sign

    • The diff(x) function computes the time derivative of variable x, representing how x changes over time in dynamic models. The sign(x) function returns the sign of x: -1 if negative, 0 if zero, and 1 if positive.

  • int, lim, rnd, disn, disc

    • The int(x) function returns the integer part of a number by truncating its decimal portion. The lim(x, a, b) function constrains the value of x within the range [a, b], returning a if x is too small and b if it's too large. The rnd(x) function rounds the number x to the nearest integer. The disn(mean, stddev) function generates a random number from a normal (Gaussian) distribution with the given mean and standard deviation. The disc(a, b) function produces a uniformly distributed random number between the values a and b.

    An example of using this functions is given below.

    //Empty model Model [type = NL, name = "examle trig fun work with num", domain=real, eps=1e-6, method=NR]: Vars [out=true]: result_lim Params: value=0.5 NLEs: result_lim = lim(12, 0, 10) end //end of model

Complex functions

The following functions are supported exclusively in models that involve complex numbers:

  • conj

    • This function returns the complex conjugate of the complex number z.

  • cabs

    • This function returns the magnitude (modulus) of the complex number z.

    An example of using this functions is given below.

    Model [type=NL domain=cmplx name="example_cabs_fun"]: Vars [conj=false out=true]: y // Variable name and initial value (optional) Params: z=2+5i NLEs: y=cabs(z) // Second nonlinear equaiton of the model end
