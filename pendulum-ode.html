1.3. Pendulum Model Described by Ordinary Differential Equations

Task Overview

The task involves the numerical solution of a system of ordinary differential equations (ODEs) to analyze the dynamic behavior of a physical pendulum model. The focus is on calculating and graphically presenting the time evolution of the angular displacement (theta) and angular velocity (omega). The model enables monitoring the influence of various physical system parameters, such as pendulum length, mass, gravity, and friction coefficient, on the overall system behavior.

Task objective:

The objective is to demonstrate the dynamic variation of key system variables—angular displacement and angular velocity—over time, while analyzing their interdependence and dependence on physical parameters. This facilitates a deeper understanding of the fundamental principles governing oscillatory systems and their behavior under ideal and real conditions.

Program Analysis:

Definition of the analyzed model:

Model [type=ODE name="Pendulum using direct ODE solver"]:

type = "ODE" – pecifies that the model solves ordinary differential equations.

name = "Pendulum using direct ODE solver" – esignates the model name.

Block Vars:

Vars [out=true]: theta=pi/6 omega=0

Block Params

Params: l=1 m=1 g=9.81 k=0

The functions of the Vars and Params blocks were explained in example 1.

Block ODEs:

ODEs: theta'=omega omega'=-g*sin(theta)/l-k*omega/(m*l)

This block is used to define ordinary differential equations that mathematically describe the behavior of the analyzed system. In this specific case, it represents the pendulum model.

The keyword end signifies the termination of the model definition

end // end of Model
