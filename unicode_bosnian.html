This section presents a Unicode example model on the dTwin platform that uses Bosnian letters with diacritical marks in variable and parameter names.

Unicode Example Model – Bosnian Letters

The model, titled "Unicode demo", demonstrates that users can define variable and parameter names using localized Bosnian characters such as "š" and "č".

Model Description

The model defines a system of linear equations describing the relationship between fruit quantities (in kilograms) and their associated costs (in monetary units). Variables šljiva (plum) and kruška (pear) represent the amount of each fruit, while parameters Trošak1 and Trošak2 represent total expenses.

Just like the previous Unicode examples using Chinese and Russian characters, this model shows that the dTwin platform fully supports the use of natural language characters. This capability makes model writing more intuitive and enhances readability — especially important in multilingual or localized environments.

Conclusion

This example confirms that dTwin supports full Unicode input, including localized Bosnian characters such as š, č, and ć. This ensures that users can model systems using terminology from their own language, improving clarity and usability — especially in educational and applied engineering environments.
