This section provides another example of Unicode support in the dTwin platform, this time using Russian (Cyrillic) letters in variable and parameter names.

Unicode Example Model – Cyrillic (Russian)

This model, titled "Unicode demo", can be found on the dTwin platform under the Unicode example section. It demonstrates that users can define variables and parameters using the Cyrillic alphabet.

Model Description

As in the previous Unicode model (with Chinese characters), this model describes a system of linear equations that links quantities of fruit (in kilograms) to their total costs (in monetary units). The variables слива (plum) and груша (pear) represent the amounts of each fruit, while the parameters Расход1 and Расход2 correspond to total expenditures.

The purpose of this model is to showcase that mathematical modeling in dTwin can be adapted to different languages and scripts. This improves readability and allows users to model problems in their native language, supporting broader use in educational, international, and localized applications.

Conclusion

This example highlights dTwin’s Unicode support across different languages and scripts. The use of Russian characters such as слива, груша, Расход1, and Расход2 illustrates the platform’s ability to handle Cyrillic identifiers naturally. This approach supports intuitive and inclusive modeling workflows, especially useful in multilingual engineering and educational environments.
