2.4. Earthquake linearized model

Task Overview

The Quake linearized model represents a simplified, linearized version of the "Cart-Pendulum" mec- hanical system, used to analyze the system’s response to external disturbances such as short-term earthquakes. This model introduces the approximation of small displacements and small angles, allowing the substitution of nonlinear functions like sine and cosine with their linear equivalents (e.g., sin(j) ≈ j , cos(j) ≈ 1 ). This significantly simplifies the mathematical formulation, enabling faster simulation and analysis of the system.

Task objective:

The objective of the simulation is to analyze the dynamic response of the system to this excitation and examine the effects of damping and stiffness parameters on system stability.

Program Analysis:

Definition of the analyzed model:

Model [type=DAE name="Quake linearized" domain=real method=RK4 eps=1e-5]:

type = "DAE" – the system uses differential-algebraic equations.

name = "Quake linearized" – model name.

domain = "real" – specifies that all variables and parameters are in the real domain.

eps = 1e-5 – specified solution accuracy.

method = "RK4" – the fourth-order Runge-Kutta method is used for solving.

Block Vars:

Vars [out=true]: x = 1 v = 0. θ = 0.5 ω = 0.

Block Params

Params: Ω = 2*pi*0.3 M=10 m=1 L=1 c_1=2 c_2=1 k=2 F_0=3 F_ext = F_0 [out=true] //Extern force mult = 1/(m*L^2*M) g=9.81

The functions of the Vars and Params blocks were explained in example 1.1.

Block ODEs:

ODEs: x' = v v' = mult * (m*L^2*(F_ext-c_1*v-k*x) + m*L*(c_2*ω + m*g*L*θ)) θ' = ω ω' = mult * (-m*L*(F_ext-c_1*v-k*x) - (M+m)*(c_2*ω + m*g*L*θ))

The function of the ODEs block was explained in example 1.3. It contains the ordinary differential equations that describe the analyzed system.

Block PostProc:

PostProc: if t < 1: //Limit extern force to be active in first second only F_ext=F_0 * cos(Ω * t) else: F_ext=0 end
The block has the same function as in the previous task.

The keyword end signifies the termination of the model definition

end // end of Model

The model enables comparison with nonlinear versions of the same system, which is useful for evaluating the validity of linear approximations in modeling mechanical systems subjected to real excitations.
