5.3. State Estimation (Gauss–Newton Normal Equations)

Task Overview:

This task implements a state estimation algorithm in a power system using the Weighted Least Squares (WLS) method. The system consists of three buses, and the goal is to estimate bus voltage magnitudes and phase angles using available measurements of active/reactive power flows, injections, and voltage magnitudes.

Task Objective:

The objective is to perform state estimation in a 3-bus power system by minimizing the weighted sum of squared measurement errors. The estimation is based on measured power flows, injections, and voltages. The WLS algorithm adjusts the state variables to best fit the measurements while accounting for their uncertainty.

WLSEs Block (Weighted Equations)

WLSEs: [w=w_br] v_1^2*g_12 - v_1*v_2*(g_12*cos(...) + b_12*sin(...)) = P_12 ... [w=w_v] v_1 = V_1 [w=w_v] v_2 = V_2

Each equation has an associated weight that reflects its measurement reliability.

Conclusion:

The WLS model provides a robust framework for state estimation by integrating different types of measurements with varying levels of accuracy. It gives a realistic insight into the network's operational state and can be used for diagnostics, control, and real-time monitoring.
