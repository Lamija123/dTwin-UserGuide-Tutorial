1.4. Pendulum Model Described by Differential-Algebraic Equations

Task Overview

This task involves the modeling and simulation of a physical system representing an ideal mathe- matical pendulum using a differential-algebraic equation (DAE) approach. The model is based on the numerical solution of a system of differential equations describing the translational motion of a mass in two dimensions, coupled with the introduction of a nonlinear constraint ensuring a constant rope length. The simulation graphically and numerically illustrates the temporal changes in position, velocity, and tension force within the system.

Task objective:

The objective is to analyze the dynamic behavior of the pendulum in a plane by utilizing a differential- algebraic model.

Program Analysis:

Definition of the model under consideration:

Model [type=DAE domain=real method=RK2 name="PendulumModelUsingDAE"]:

type = "DAE" – the system utilizes differential-algebraic equations (DAE).

name = "PendulumModelUsingDAE" – the model’s designated name.

domain = "real" – specifies that all variables and parameters belong to the real domain.

method = "RK2" – indicates that the second-order Runge-Kutta method is employed for solving the system.

Block Vars:

Vars [out=true]: // output all variables for plotting x=0.6 // Variable name and initial value (optional) v_x=0 // Variable name and initial value (optional) y=sqrt(l^2-x^2) // "0.998749218" Variable name and initial value (optional) v_y=0 // Variable name and initial value (optional) F=m*g*y/l // "9.6" Variable name and initial value (optional)

Block Params

Params: // default integration step l=1 // Parameter name and its value m=1 // Parameter name and its value g=9.81 // Parameter name and its value k=0 // koeficijent otpora zraka v=0 [out=true] // brzina

The functions of the Vars and Params blocks were explained in example 1.

Block ODEs:

ODEs: x'=v_x // First state variable v_x'=-F*x/(m*l)-k*v_x/m // Second state variable y'=v_y // 3rd state variable v_y'=g-F*y/(m*l)-k*v_y/m // 4th state variable

Block NLEs:

NLEs: x^2+y^2=l^2 // First nonlinear equaiton of the model

The functions of the NLEs and ODEs blocks were explained in examples 2 and examples 3, respectively. Here, differential and algebraic nonlinear equations describing the analyzed system are entered.

Block PostProc:

PostProc: v=sqrt(v_x^2+v_y^2)

This block enables post-processing calculations of quantities, in this case the system’s velocity.

The keyword end signifies the termination of the model definition

end // end of Model
