3.2. RLC circut with Sine Generator

Task Overview

The task involves the mathematical modeling of a series RLC circuit powered by a sinusoidal voltage of frequency 50 Hz, simulating real operating conditions in power systems. The system is described using a set of ordinary differential equations (ODEs), which model the behavior of the current through the inductor (i_L), the voltage drop across the inductor (u_L), and the voltage across the capacitor (u_c) in the time domain.

Task objective:

To formulate a mathematical model of a series RLC circuit with sinusoidal excitation voltage. To monitor changes in key physical quantities (i_L, u_c, u_L) over time. To visually present the oscillatory behavior of the system and analyze the impact of frequency, phase shift, and components (R, L, C) on the dynamic response.

Program Analysis:

Definition of the model under consideration:

Model [type=ODE domain=real name="Serial RLC With Sine Voltage Gen"]:

type = ”ODE” – the model solves ordinary differential equations.

name = ”Serial RLC With Sine Voltage Gen” – model name

domain = ”real” – defines that all variables and parameters are in the real domain.

Block Vars:

Vars [out=true]: u_c=0 i_L=0

Block Params

Params: t=0 f=50 omega=2*pi*f phi=pi/4 R=5 L=0.012 C=8e-4 u_max=230*sqrt(2) // phase to ground EU LV u_g=u_max*sin(omega*t+phi) [out=true] u_L=u_g [out=true]

The functions of the Vars and Params blocks were explained in example 1.

Block ODEs:

ODEs: i_L'=(u_max*sin(omega*t+phi)-u_c-R*i_L)/L u_c'=i_L/C

The Block a ODEs function is explained in examples 3. Here, differential and algebraic nonlinear equations describing the analyzed system are entered.

Block PostProc:

PostProc: u_g = u_max*sin(omega*t+phi) u_L = u_g-u_c-R*i_L

After simulating the main variables, the excitation and the voltage across the coil (L) are calculated, enabling additional visualization.

The keyword end signifies the termination of the model definition

end // end of Model

The first and second RLC circuit models are based on the same physical components — resistor (R), coil (L), and capacitor (C) — but differ significantly in the type of excitation, which directly affects the system's dynamic characteristics, simulation objectives, and application scope. Different types of excitation lead to varying system responses. Step excitation is ideal for analyzing switching dynamics, while sinusoidal excitation is suitable for frequency analysis and system stability.
