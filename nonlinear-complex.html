1.2. Nonlinear Equations in the Real Domain

Task Overview

This task addresses solving a system of nonlinear equations within the complex domain. The model comprises two complex variables and two nonlinear equations involving addition and multiplication of complex numbers.

Task objective:

The objective is to determine the variable values that satisfy the given equations.

Program Analysis:

We begin by defining the desired model.

Model [ type = NL domain = cmplx name = " CmplxDemo " ]:

The desired model properties are specified as follows::

type = "NL" - the model contains nonlinear equations.

name = "CmplxDemo" - The model is named "CmplxDemo".

domain = cmplx" - All expressions are defined within the complex domain.

Block Vars

Vars [conj=false out=true]: x=e+1i*e y=0

This block defines the model variables.

The keyword Vars indicates the start of the variable list. The optional attribute [out = true] specifies whether the variables will be included in the simulation results. The optional attribute [conj=false] disables automatic generation of conjugate variables when set to false. The variable x is initialized with the complex number (e + i * e), and variable y is initialized to zero.

Block Params

Params: z=2+5i

As explained in example 1, this section is used to define the constants employed in the model equations.

Block NLEs

NLEs: x+y-1+2i=z x*y=4i

This section specifies the nonlinear equations of the system to be solved numerically.

The keyword end signifies the termination of the model definition

end // end of Model
