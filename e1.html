5.8. Power Flow: 3 Nodes and PQ Load

Power Systems - Complex

Task Overview:

This task presents a power flow analysis using complex phasor representation for a 3-node power network. The network includes:

  • Node 1: Slack node with fixed voltage v₁ = 1 + 0i
  • Node 2: Zero-injection (no net power injection)
  • Node 3: PQ node with known active and reactive power consumption

Task Objective:

The goal is to calculate the voltage magnitudes and angles at nodes 2 and 3 using the complex Newton-Raphson method. The model enforces Kirchhoff’s current law at node 2 and power injection constraint at node 3.

Program Analysis:

Model Definition

Model [type=NL domain=cmplx eps=1e-6 name="PF for 3 nodes (second node is Zero Injection)"]:
  • type = "NL" – nonlinear model
  • domain = "cmplx" – complex-valued variables and equations
  • eps = 1e-6 – convergence threshold

Variables

  • v_2, v_3 – complex voltage phasors at nodes 2 and 3

Parameters

  • v_1 = 1+0i – reference voltage (slack node)
  • Impedances: z₁₂, z₂₃, z₂₄ with admittances yij = 1/zij
  • S₃ = –1 – 0.3i – complex power load at node 3

Nonlinear Equations (NLEs)

  • Kirchhoff’s current law at node 2: y₂₂v₂ – y₂₁v₁ – y₂₃v₃ = 0 and its conjugate
  • Power constraint at node 3: v₃ × conjugate(i₃) = S₃, where i₃ = y₃₃v₃ – y₂₃v₂

Observations

  • Convergence achieved in 3 iterations
  • Voltage magnitude drops as power flows from node 1 to PQ node 3
  • Angle φ₃ stabilizes around –3.14 radians (≈ –180°), typical for inductive load

Conclusion

The complex-domain power flow model computes system state using Newton-Raphson iterations. With just 3 iterations and physically consistent voltages and angles, the method is both mathematically stable and accurate. Use of conjugate symmetry improves robustness, especially in lossy or unbalanced networks.
