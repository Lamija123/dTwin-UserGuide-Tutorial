Download and installation

The ModelSolver application and the dTwin platform can be downloaded from the website. On the download page, users should select the appropriate operating system and proceed with the appropriate installation steps.

In addition, within the ZIP archive obtained through this installation, users can find several models and examples that may serve as useful references when working with the dTwin platform.

A brief overview of the contents of the ZIP archive is provided below:

  • dTwin is a GUI (Graphical User Interface) application for digital twinning.
  • ModelSolver is a terminal-based application that requires user input of parameters.
  • The Models folder contains several models in .dmodl and .vmodl formats.

For a successful installation, it is recommended to first read the contents of the file 01.ReadMe.txt. A detailed description of the installation and uninstallation process is outlined in the following steps:

  1. To install the application, run the installation executable file (setup.exe) by double-clicking it.
  2. If the application does not function after installation, it is necessary to install the C++ Redistributable Packages by running the file VC_redist.x64.exe.
  3. To uninstall the application on Windows, use the "Add or Remove Programs" option.
