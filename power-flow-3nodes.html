5.1. Power Flow: 3 Nodes and PQ Load

Power Systems - Real

Task Overview:

This task demonstrates solving a power flow problem in a three-bus power system using a real-valued nonlinear model. Bus 1 is the slack bus with fixed voltage magnitude and phase angle, bus 2 is a zero-injection node (no power inflow or outflow), and bus 3 is a PQ load bus with specified active and reactive power consumption. The model uses known admittance values and phase angles between buses to formulate the system of nonlinear equations representing the power balance.

Task objective:

The objective is to analyze the voltage magnitudes and phase angles at buses 2 and 3 using a real-domain nonlinear mathematical model. Specifically, the goal is to solve the system of nonlinear equations representing active and reactive power balance at each bus, based on given network parameters, and determine the unknown voltage variables.

Program Analysis:

The program is defined as a nonlinear model (type=NL) in the real domain (domain=real), with a convergence tolerance of eps = 1e-6.

Variables

  • phi_2, v_2 – phase angle and voltage magnitude at bus 2
  • phi_3, v_3 – phase angle and voltage magnitude at bus 3

Parameters

  • v_1 = 1, phi_1 = 0 – known voltage magnitude and angle at the slack bus
  • aY, theta – admittance values and their phase angles
  • P3_inj = -1, Q3_inj = -0.3 – active and reactive power injection at bus 3
