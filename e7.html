5.14. State estimation with S-V (P-V) curve limit

Task Overview:

This task evaluates the ability of the state estimator to track the loadability limit of a power system by using synthetic measurements generated via a submodel. The test mimics a transfer capability study by gradually increasing the power demand at node 3 and applying noisy measurements into the state estimation process.

Task Objective:

The objective is to simulate the S-V (or P-V) curve using a weighted least squares (WLS) state estimation model with equality constraints, where:

  • Sensor noise is added to voltage magnitudes and power injections,
  • The Zero-Injection (ZI) condition is enforced on node 2,
  • The estimation is repeated with incremental load increase at node 3,
  • Estimation fails beyond the critical point, indicating loadability limit.

Program Analysis:

The model is composed of:

  1. Submodel which generates noisy synthetic measurements based on a nonlinear PF formulation,
  2. Main WLS estimation model which uses these measurements to estimate the state under constraints.
Model [type=WLS reInit=true eps=1e-5 maxIter=50 domain=cmplx name="SE EC With PF SubModel"]:

Key Elements

  • Slack node: Node 1 (fixed angle and magnitude),
  • PQ node: Node 3 (load increases during repeats),
  • ZI node: Node 2 (no net injection),
  • Voltage magnitude and power injection measurements are disturbed using Gaussian noise.

Monte Carlo + CPF Approach

  • The submodel generates a noisy snapshot of the grid based on a known power injection S3,
  • The main model then estimates v2, v3, and state variables from noisy measurements,
  • The process is repeated by increasing S3 iteratively until convergence fails.

Figure 5.14.1: S-V (P-V) curve for node 3 voltage magnitude

Observations

  • The curve shows a decreasing trend in \( v_3 \) as load increases, typical for P-V analysis.
  • At higher load levels (above approx. 5 p.u.), noise increases and convergence becomes erratic or fails.
  • The estimator is stable in early iterations but shows visible degradation near the limit.

Conclusion:

The experiment demonstrates the integration of Monte Carlo-based noisy measurement generation and CPF-style loading in a state estimation framework. It provides a practical means of identifying system loadability limits under uncertainty. The state estimator correctly follows the S-V curve trend and signals instability as expected when approaching critical points.
